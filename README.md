# F1 Driver Management System

Веб-приложение для управления данными чемпионата Формулы-1. Система позволяет отслеживать информацию о пилотах и командах, а также визуализировать текущее положение в личном зачете и Кубке конструкторов в реальном времени.

## Основные функции

* **Управление командами и пилотами**: Полный контроль над составом участников, включая регистрацию номеров пилотов, их зарплат и технических характеристик болидов.
* **Интерактивный дэшборд**: Автоматический расчет общего количества очков, подиумов и уникальной метрики.
* **География чемпионата**: Управление географией гонок с привязкой конкретных Гран-при к существующим автодромам и трассам.
* **Детальная статистика пилотов**: Персональные страницы гонщиков с полной историей их выступлений и ключевыми показателями (победы, подиумы, очки).

## Технологический стек

* **Backend**: Python 3.10+, Flask
* **Database**: PostgreSQL
* **Frontend**: Bootstrap 5, HTML5, CSS3

## Инструкция по запуску

### 1. 1. Клонирование репозитория
```bash
git clone https://github.com/matorafull/f1-driver-management-system
cd f1-management-system
```

# Настройка окружения
```bash
python -m venv venv
# Для Windows:
venv\Scripts\activate
# Для Linux/Mac:
source venv/bin/activate

pip install flask psycopg2-binary
```

### 3. Настройка базы данных
1.  Создайте базу данных в PostgreSQL с названием *f1_racing_db*
2.  Откройте инструмент запросов (Query Tool) и выполните команды из файла init_db.sql для создания таблиц.

3.  При первом запуске SQLAlchemy автоматически создаст все необходимые таблицы на основе описанных моделей.

### 4. Запуск проекта
Выполните следующую команду из корня проекта:
```bash
python app.py
```

## Структура базы данных

* `circuits` — справочник гоночных трасс и автодромов.
* `teams` — данные о командах и их бюджетных лимитах.
* `drivers` — реестр лицензий пилотов и их контрактов.
* `cars` — технические параметры болидов, привязанных к командам.
* `races` — календарь запланированных событий сезона.
* `race_results` — архив финишных протоколов и набранных очков.
